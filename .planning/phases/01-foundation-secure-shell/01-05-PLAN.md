## Objective

Implement the custom frameless title bar (D-12/D-13) and window size persistence (D-14). This involves:

1. Switching `BrowserWindow` to frameless (`titleBarStyle: 'hidden'` on macOS, `frame: false` on Win/Linux)
2. Populating the empty `windowRouter` with `minimize`, `maximize`, `close`, `isMaximized` tRPC procedures
3. Building the `TitleBar` React component with drag region, app name, campaign name (when inside a campaign), and native controls for Win/Linux
4. Mounting `TitleBar` above all routes in `App.tsx`
5. Persisting and restoring window `width`, `height`, `x`, `y` via `electron-store`

Purpose: Closes part of Gap 1 (SC-3 visual completeness — the title bar is the top chrome above the split-panel shell) and satisfies FOUND-01 (packaged build launches correctly with correct window chrome).

Output:
- `src/main/index.ts` — updated BrowserWindow to frameless + window bounds persistence
- `src/main/trpc/routers/window.ts` — minimize/maximize/close/isMaximized procedures
- `src/renderer/src/components/TitleBar.tsx` — new title bar component
- `src/renderer/src/App.tsx` — TitleBar mounted above routes
- `src/renderer/src/stores/windowStore.ts` — Zustand store for campaign name in title bar
