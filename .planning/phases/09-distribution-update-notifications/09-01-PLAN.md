<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GitHub API → main process | Untrusted remote JSON (tag_name, html_url, prerelease) crosses into the app over HTTPS |
| renderer → main (tRPC) | dismissUpdate input (version string) crosses from untrusted renderer into the persisted store |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-03 | Tampering | updateChecker.ts fetch of GitHub Releases API | mitigate | HTTPS-only URL constant; html_url is returned but NOT acted on here — Plan 02 preload validates it before shell.openExternal. No cert pinning (low-value, public data). |
| T-09-04 | Tampering | dismissUpdate tRPC input | mitigate | Zod `z.object({ version: z.string() })` validates at procedure boundary before appPrefsStore.set |
| T-09-05 | Denial of Service | malformed tag_name (NaN injection) | mitigate | isNaN guard on parsed semver parts returns the safe fallback { available: false } instead of crashing |
| T-09-06 | Denial of Service | GitHub API hang / rate limit | mitigate | AbortController 5s timeout; non-ok responses return fallback; check is fire-and-forget (never blocks startup, D-04) |
</threat_model>
