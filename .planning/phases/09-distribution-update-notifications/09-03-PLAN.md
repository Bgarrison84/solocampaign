## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-10 | Spoofing | unsigned installers (Windows/macOS) | accept | D-01: ship unsigned for v1.0; SmartScreen/Gatekeeper bypass documented in release notes (no certs procured). Revisit when signing is added (deferred). |
| T-09-11 | Tampering | concurrent matrix Release uploads | mitigate | Aggregator pattern: matrix jobs only upload-artifact; a single publish job (needs: [build]) creates the Release once — avoids the body-update race (RESEARCH Pitfall 2) |
| T-09-12 | Elevation of Privilege | GITHUB_TOKEN scope in publish job | mitigate | publish job declares `permissions: contents: write` (least privilege — only Release write, not full repo) |
| T-09-13 | Tampering | native module arch mismatch | mitigate | Per-platform `npm_config_arch` (arm64 for macos-latest) ensures the .node binary matches the runner's Electron arch (RESEARCH Pitfall 1) |
| T-09-SC | Tampering | npm/pip/cargo installs | accept | No new npm packages this phase; CI actions (softprops/action-gh-release@v2, actions/upload-artifact@v4, actions/download-artifact@v4) are RESEARCH-audited and Approved (Package Legitimacy Audit) |
